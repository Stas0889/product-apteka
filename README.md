# Продуктовая аптечка

Статическая версия мини-гайда для GitHub Pages.

## Материалы

- `product-apteka/` - продуктовая аптечка
- `clean-blood/` - чистая кровь
- `parazity-upravlyayut-vami/` - тест «Паразиты управляют вами прямо сейчас?»

## Публикация

Загрузите содержимое этой папки в корень GitHub-репозитория:

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`
- `assets/`
- `fonts/`

В настройках GitHub Pages выберите публикацию из ветки `main` и папки `/root`.

Если хотите хранить весь проект в репозитории, переименуйте готовую папку сайта в `docs/` и выберите публикацию из `/docs`, либо используйте GitHub Actions. Самый простой вариант - публиковать именно содержимое готовой папки сайта как корень репозитория.
